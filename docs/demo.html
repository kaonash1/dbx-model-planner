Offline MVP

Databricks model fit planning without the platform ceremony.

This page is a static visual walkthrough of the current CLI. The tool is intentionally simple: inventory, model-to-compute fit, compute-to-model fit, cost estimates, and a deployment hint.

2. Model Fit

Given a model, which compute fits?

PYTHONPATH=src python3 -m dbx_model_planner model fit mistral-7b-instruct

Recommended compute for mistralai/Mistral-7B-Instruct-v0.3:
Standard_NC6s_v3 (borderline, int8).
- Standard_NC6s_v3: borderline, int8, est mem 14.8 GB
- Standard_D3_v2: unlikely, int8, est mem 14.8 GB
3. Compute Fit

Given a GPU node, what model ranges are realistic?

PYTHONPATH=src python3 -m dbx_model_planner compute fit Standard_NC6s_v3

Compute Standard_NC6s_v3 can safely or borderline run 5 of 5 evaluated models.
Model family ranges:
- embedding: 0.3B to 0.3B
- llm: 7.0B to 8.0B
- vlm: 2.0B to 2.0B

Example models:
- BAAI/bge-large-en-v1.5: safe, fp16, est mem 1.6 GB
- intfloat/e5-large-v2: safe, fp16, est mem 1.6 GB
- meta-llama/Llama-3.1-8B-Instruct: safe, int4, est mem 11.9 GB
- Qwen/Qwen2-VL-2B-Instruct: safe, int4, est mem 3.8 GB
- mistralai/Mistral-7B-Instruct-v0.3: borderline, int8, est mem 14.8 GB
4. Cost

Apply VM, DBU, discount, and VAT.

  Standard_NC6s_v3 --vm-rate 3.25 --dbu-rate 0.75

5. Deploy Hint

Generate the next safe step, not orchestration.

PYTHONPATH=src python3 -m dbx_model_planner deploy plan qwen2-vl-2b-instruct

Stage Qwen/Qwen2-VL-2B-Instruct into a Unity Catalog volume.
